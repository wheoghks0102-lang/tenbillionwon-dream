
# 시나모롤 코딩 웹사이트 프로젝트

## 1. 개요

이 프로젝트는 사용자의 상호작용을 통해 시나모롤이 코딩을 하고, 그 결과가 6명의 캐릭터에게 영향을 미치는 상황을 시뮬레이션하는 웹 애플리케이션입니다. 귀여운 시나모롤 테마를 기반으로 하여 사용자에게 재미와 즐거움을 선사하는 것을 목표로 합니다.

## 2. 디자인 및 기능 명세

### 가. 핵심 기능
- **시나모롤 코딩 시뮬레이션**: 사용자가 버튼을 클릭하면 시나모롤이 코딩을 시작하는 애니메이션 또는 텍스트 효과가 나타납니다.
- **랜덤 이벤트 시스템**: 코딩 후, 미리 정의된 여러 이벤트 중 하나가 랜덤하게 발생하며, 이벤트 내용은 텍스트로 출력됩니다.
- **캐릭터 상태 변화**: 발생한 이벤트에 따라 6명 캐릭터(심우성, 채의진, 조윤혜, 조대환, 최준, 전유희)의 상태(체력, 포만감, 재화)가 실시간으로 변화하고, 이는 UI에 즉시 반영됩니다.

### 나. 화면 구성
- **전체 레이아웃**: 화면은 크게 좌측의 '시나모롤 코딩 구역'과 우측의 '캐릭터 상태 구역'으로 나뉩니다.
- **좌측 (시나모롤 코딩 구역)**:
    - 상단: `coding cinnamonroll.jpg` 이미지 표시
    - 중단: 코딩 결과가 출력되는 텍스트 박스
    - 하단: "코딩하기!" 버튼
- **우측 (캐릭터 상태 구역)**:
    - 6명의 캐릭터 카드가 세로로 나열됩니다.
    - 각 카드에는 캐릭터 이름, 체력 바, 포만감 바, 그리고 현재 보유한 재화가 표시됩니다.

### 다. 캐릭터 상세
- **초기 상태**:
    - 체력: 70/100
    - 포만감: 70/100
    - 재화: 0원 / 1,000,000원
- **상태 표시**: 체력과 포만감은 프로그레스 바로 시각화하며, 재화는 텍스트로 표시합니다.

### 라. 디자인
- **테마**: 시나모롤 캐릭터의 정체성을 살린 파스텔 톤(하늘색, 흰색, 분홍색)을 메인 컬러로 사용합니다.
- **UI 요소**: 모든 UI 요소(버튼, 카드, 텍스트 박스)는 둥근 모서리를 사용하여 부드럽고 귀여운 느낌을 강조합니다.
- **폰트**: 가독성이 좋고 귀여운 느낌의 'M PLUS Rounded 1c' 폰트를 Google Fonts를 통해 적용합니다.
- **애니메이션**: 버튼 클릭이나 상태 변화 시 부드러운 전환 효과를 추가하여 시각적 즐거움을 더합니다.

## 3. 개발 계획

### 1단계: HTML 구조 설계 (`index.html`)
- 전체적인 레이아웃을 잡고, 좌측과 우측 구역을 `div`로 나눕니다.
- 시나모롤 이미지, 코드 출력창, 액션 버튼의 기본 HTML 요소를 배치합니다.
- 6명 캐릭터의 정보를 표시할 카드 구조를 생성합니다.

### 2단계: CSS 스타일링 (`style.css`)
- 시나모롤 테마에 맞는 색상, 폰트, 배경을 적용합니다.
- Flexbox 또는 Grid를 사용하여 2단 레이아웃을 구현합니다.
- 각 UI 요소(이미지, 버튼, 카드, 프로그레스 바)에 디자인 명세에 따른 스타일을 적용합니다.

### 3단계: JavaScript 로직 구현 (`main.js`)
- 캐릭터 데이터를 객체 배열로 관리합니다.
- 페이지 로드 시 캐릭터의 초기 상태를 UI에 렌더링하는 함수를 작성합니다.
- "코딩하기!" 버튼에 클릭 이벤트 리스너를 추가합니다.
- 버튼 클릭 시 랜덤 이벤트를 생성하고, 이벤트 내용에 따라 캐릭터의 상태 데이터를 업데이트하는 로직을 구현합니다.
- 변경된 데이터를 UI에 다시 렌더링(상태 바 너비 조절, 텍스트 업데이트 등)하는 함수를 작성합니다.
